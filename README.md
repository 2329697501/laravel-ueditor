# Baidu UEditor for Laravel

A laravel package of ueditor,which is baidu's open source WYSIWYG editor

## Install

Via Composer

``` bash
$ composer require "ender/laravel-ueditor": "dev-master"
```

## Usage

``` php
upload route : /ueditor/server
```
